Detailed Description

template<class Key, class Data, class CacheKey>
class DiffPTData< Key, Data, CacheKey >

Diff points-to data with cached information This is an optimisation version on top of base points-to data. The points-to information is propagated incrementally only for the different parts. CahcePtsMap is an additional map which maintains cached points-to.

Detailed Description

template<class Key, class Data>
class DFPTData< Key, Data >

Data-flow points-to data structure, points-to is maintained for each program point (statement) For address-taken variables, every program point has two sets IN and OUT points-to sets For top-level variables, their points-to sets are maintained in a flow-insensitive manner via getPts(var).
